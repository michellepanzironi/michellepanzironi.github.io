What I've Been up to Lately

Clever Music is a single-page, full-stack web application inspired by popular lyrics website Genius.com. It is built with Ruby on Rails and PostgreSQL on the backend, and React.js with Redux framework for the frontend.

ORM Lite is an Object Relational Mapping tool that is inspired by Rails ActiveRecord. As an exercise in metaprogramming, ORM Lite uses Ruby to generate SQL query code and provides CRUD, search and association functionalities.

BeSparkly is a match-3 game in the style of BeJeweled or Candy Crush. It is built using vanilla Javascript, HTML5 and CSS. Users switch gems around the board and the game removes rows of 3 or more matching jewels, and generates new jewels to place at the top of the grid.
